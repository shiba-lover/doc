            # 阿良Api-接口开放平台的设计与开发

            # 项目介绍

            # 1、项目背景

            1、前端开发需要用到后端提供的接口（比如Swagger）
2、第三方开发平台（高德开发平台、讯飞星火api）

            # 2、项目功能介绍

              做一个提供API接口调用的平台，用户可以注册登录、开通接口调用权限。用户可以调用接口，并且统计调用次数。管理员可以发布接口、下线接口、接入接口，
以及可视化接口的调用情况等等。

            # 3、项目安全性考虑

            防止攻击、限制接口调用、统计调用次数、计费、流量保护、第三方api接入

            # 业务流程

            1、管理员提供后台管理系统，进行接口管理（包括发布接口、下线接口、接口设置）。
2、用户通过前台，访问接口文档、在线调用接口。
3、使用API网关（kong、nginx、geteway）,处理统计次数、计费、接口保护、鉴权、日志、跨域等业务，防止用户直接访问数据库。
4、开发SDK提供给开发者使用。

            # 技术选型

            # 前端：

            Ant Design Pro
React
Ant Design Pro components
Umi
Umi Request

            # 后端：

            Springboot
Springboot Starter（SDK开发）

            # 需求分析

            1、管理员可以对用户信息、接口信息进行管理
2、用户可以访问前台，查看接口信息
3、用户可以调用接口，但需要进行API签名认证。

            # 数据库设计

            1、用户表

            字段 数据类型 备注
            id bigint 数据id
            userAccount varchar(256) 登录名
            userPassword varchar(512) 加密密码
            unionId varchar(124) 微信开放平台id
            mpOpenId varchar(256) 微信公众号Id
            userName varchar(256) 用户名
            userAvatar varchar(1024) 头像地址
            userProfile varchar(512) 用户简介
            userRole varchar(256) 用户角色
            createTime datetime 创建时间
            updateTime datetime 更新时间
            status tinyint 账户状态
            isDelete tinyint 是否删除

            2、接口信息表 interface_info

            字段 数据类型 备注
            id int id
            name varchar 接口名称
            description varchar 描述
            url varchar 接口地址
            method varchar 请求类型
            requestHeader varchar 请求头
            respondHeader varchar 响应头
            status int 接口状态，0-关闭，1-开启
            userId bigint 用户id
            createTime datetime 创建时间
            updateTime datetime 修改时间
            isDelete int 是否删除，默认0

            3、访问键表 access_key

            字段 数据类型 备注
            id bigint 数据id
            accessKeyId varchar(256) 访问键id
            accessKeySecret varchar(256) 访问键密码
            userId bigint 用户id
